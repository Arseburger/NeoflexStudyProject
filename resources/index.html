Welcome

This is a Calculator app made for Node.JS course by Neoflex

To use this app simply type /calculator/:num1/:operation/:num2 (e.g. /calculator/3/times/4), where
'num1' and 'num2' parameters are the first and second operands respectively, and
'operation' is the mathematical operation, which would be applied to those two operands.

Available operations are:

  • Addition (/plus);
  • Substraction (/minus);
  • Multiplication (/times);
  • Division (/by).

    • This app handles following errors:

  • Invalid user input (values are not numbers);
  • Invalid user input (wrong operation name);
  • Division by zero.
